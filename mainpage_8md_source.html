4 ## Introduction
5 
6 A MAVLink router and firewall. It can connect over serial and UDP with 2 or
7 more MAVLink endpoints such as autopilots, ground control software, loggers,
8 image capture systems, etc. MAVLink packets will be routed when they are
9 addressed and can be filtered based on source system/component and message type.

24 ## Install
25 
26 In order to compile you will need the following packages:
27 
28 * GCC 7+ or Clang 5+ (needed for C++17 support)
29 * [CMake](https://cmake.org/)
30 
31 `mavtables` can be easily installed using the standard procedure of
32 ```
33 $ make
34 # make install
35 ```
36 The installation prefix is `/usr/local` by default but can be changed with
37 ```
38 $ make
39 # make PREFIX=/desire/install/path install
40 ```
